1.	在游戏开始时，解锁一个站位，新手教学后，上阵了一个英雄后，开放第二个英雄站位。
第三站位解锁时机：通关第10关
第四站位解锁时机：通关第20关
第五站位解锁时机：通关第30关
	在站位未解锁时，站位上要有个锁，点击或者拖动英雄到站位时，提示站位尚未解锁，点击时还会显示解锁要求
2.	当英雄在战斗中死亡跑开后，需要达到下一关或者此关失败后，再次从左边登场，如果战斗结果为失败时，则在黑幕过后退关重新登场
3.	替换英雄: 在战斗过程中可以对英雄进行上下阵操作，下阵过程需要播放动画英雄跑回的动画大约5s，所替换的英雄上阵需要播放从左侧走出的动画，大约10s。
4.	在上阵英雄的头像上展示英雄状态，上阵中，死亡，下阵中等标签动画（ps英雄标签处于上阵中，下阵中的时候不可进行替换，下阵等操作）
